This website is a learning exercise set by The Odin Project

The OdinProject is a full stack program of study from setting up your dev environment,
using the command line, Git & Github, through to Front and Backend.

This site is to walk us through creating a server and using it to direct users to different pages.

It will also show a custom 404 page not found page in the event of a bad request.

Please note: This excercise instructs us to spend time on the backend, and not the frontend UX.
